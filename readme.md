# AMaDaS – Research Group Website (Overview)

This page is a simple landing website for the AMaDaS research group. It presents a clean, modern layout with a top navigation bar, a main introduction section, and a footer area for contact details. The design uses a dark theme and includes light decorative elements to create a friendly, celebratory atmosphere while keeping the focus on the group’s identity and mission.

## How to run locally
This is a static website (HTML/CSS/JavaScript). You can run it locally using Python’s built-in web server:

```bash
python3 -m http.server 5500
